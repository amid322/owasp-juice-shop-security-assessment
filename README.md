# Exploit-JuiceShop
This repository documents my hands-on security testing of the OWASP Juice Shop application, a deliberately vulnerable web app used for security training.

## Objectives
- Practice real-world web application penetration testing
- Identify and exploit common web vulnerabilities
- Document findings in a professional security report format

## Skills Demonstrated
- Web application reconnaissance
- Access control testing
- Injection testing (SQLi, XSS)
- Security reporting and remediation guidance

## Scope
Testing was performed in a local lab environment against OWASP Juice Shop.

## Disclaimer
This project is for educational purposes only. All testing was conducted in a safe and controlled lab environment.
